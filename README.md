## 关于字体
根据字体公司及微软的规定，Windows 随附的字体只能在 Windows 中使用。由于我日常使用 GNU/Linux 操作系统，我无法用这些字体进行排版。如果您需要使用 Windows 随附的字体，请参考模板文件里的提示。
